# WS Chat Client

Vanilla TypeScript frontend built with Vite.

## Environment

Create local env:

```bash
cp .env.example .env.local
```

Variables:

```bash
VITE_API_PORT=4001
VITE_WS_PORT=8001
```

These must match backend `API_PORT` and `WS_PORT`.

## Commands

```bash
npm run dev
npm run build
npm run preview
npm run service:start
```

Default dev URL:

```bash
http://localhost:5173
```
